    # TCP编程

    • 特点：
      • 使用TCP协议前，先建立TCP连接，形成传输通道
      • 传输前，采用“三次握手”方式，方式可靠
      • TCP协议进行通信的两个进程：客户端、服务端
      • 在连接中可以进行大数据量的传输
      • 传输完毕，需要释放已建立的连接，效率低

    第一次写就备注的详细点。

    /*
1.编写一个服务端和一个客户端
2.服务器端在9999端口监听
3.客户端连接到服务器端，发送“hello world” ，然后退出
4.服务器端接收到客户端发送的消息，输出，并退出

要求利用字节读取
 */

// 客户端
public class Client {
    public static void main(String[] args) throws IOException {
        // 意思就是 我要连接本机的9999 端口 如果连接成功，返回socket对象
        Socket socket = new Socket(InetAddress.getLocalHost(), 9999);
        String Msg = "你好，我叫lds";


        // 连接上后，生成了socket对象，通过调用getOutputStream()
        // 得到一个 和 socket对象关联的输出流对象
        OutputStream os = socket.getOutputStream();
        // 通过输出流，写入数据到 数据通道
        os.write(Msg.getBytes());
        System.out.println("发送成功");

        // 这里设置一个结束的标记
        socket.shutdownOutput();

        InputStream is = socket.getInputStream();
        byte[] buf = new byte[1024];
        int readLen = 0;
        while ((readLen = is.read(buf)) != -1) {
            System.out.println(new String(buf, 0, readLen));
        }
        is.close();
        socket.close();
        os.close();
    }
}

// 服务端
public class Serve {
    public static void main(String[] args) throws IOException {
        ServerSocket serverSocket = null;
        try {
            // 细节：不能有其他程序在监听9999 端口
            serverSocket = new ServerSocket(9999);
        } catch (IOException e) {
            e.printStackTrace();
        }
        // 如果没有客户端连接9999 端口，程序会阻塞，等待连接
        // 如果有客户端连接，返回Socket对象，程序继续
        // serverSocket.accept() 返回多个socket（这就是可以使得多个客户端连接）
        Socket socket = serverSocket.accept();

        System.out.println("socket：" + socket.getClass());

        InputStream is = socket.getInputStream();
        byte[] buf = new byte[1024];
        int readLine = 0;
        while ((readLine = is.read(buf)) != -1) {
            System.out.println(new String(buf, 0, readLine));
        }

        // 这里设置一个结束的标记
        socket.shutdownInput();
        OutputStream os = socket.getOutputStream();
        os.write("你好呀，很高兴认识你！".getBytes());

        os.close();
        is.close();
        socket.close();

    }
}

    public class StreamUtils {

    /**
     * 将二进制文件 转化为 字节数组
     * @param is 输入流对象
     * @return 字节数组
     * @throws Exception
     */
    public static byte[] streamToByteArray(InputStream is) throws Exception {
        ByteArrayOutputStream bos = new ByteArrayOutputStream(); // 创建一个字节数组输出流对象
        byte[] buf = new byte[1024];
        int len;
        while ((len = is.read(buf)) != -1) { // 读取数据在buf中
            bos.write(buf, 0, len); // 将buf数组内容写入到bos输出流中
        }
        byte[] array = bos.toByteArray(); // 转化为新的字节数组
        bos.close();
        return array;
    }
}

    // 服务端
public class Serve {
    public static void main(String[] args) throws Exception {
        ServerSocket serverSocket = new ServerSocket(9999);
        // 形成阻塞
        Socket socket = serverSocket.accept();
        InputStream is = socket.getInputStream();

        // 主要代码部分
        String s = "";
        byte[] buf = new byte[1024];
        int len = 0;
        while ((len = is.read(buf)) != -1) {
            s += new String(buf, 0, len);
        }
        BufferedInputStream bis = null;
        if ("高山流水".equals(s)) {
            bis = new BufferedInputStream(new FileInputStream("src/" + s + ".mp3"));
        } else {
            bis = new BufferedInputStream(new FileInputStream("src/无名.mp3"));
        }
        byte[] bytes = StreamUtils.streamToByteArray(bis);
        BufferedOutputStream bos = new BufferedOutputStream(socket.getOutputStream());
        bos.write(bytes);
        socket.shutdownOutput();

        bos.close();
        is.close();
        socket.close();
    }
}
// 客户端
public class Client {
    public static void main(String[] args) throws Exception {
        Socket socket = new Socket(InetAddress.getLocalHost(), 9999);
        BufferedInputStream bis = new BufferedInputStream(socket.getInputStream());
        byte[] bytes = StreamUtils.streamToByteArray(bis);
        String s = "D:\\音乐.mp3";
        BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(s));
        bos.write(bytes);
        socket.shutdownInput();
        
        bos.close();
        bis.close();
        socket.close();
    }
}

    # UDP编程

    • 特点：
      • 类DatagramSocket和DatagramPacket【数据包/数据报】实现了基于UDP协议的网络程序
      • UDP数据报通过数据套接字DatagramSocket发送和接收，系统不保证UDP数据报一定能够安全送达目的地，也不能确定什么时候可以抵达
      • DatagramPacket对象封装了UDP数据报，在数据报中包含了发送端的IP地址和端口号以及接收端的IP地址和端口号
      • UDP协议中每个数据报都给出了完整的地址信息，因此无需建立发送方和接收方的连接
    // 接收端A
public class A {
    public static void main(String[] args) throws IOException {
        byte[] buf = null;
        // 1.创建一个DatagramSocket对象，准备在9999接收数据
        DatagramSocket socket = new DatagramSocket(9999);
        // 2.构建一个DatagramPacket对象，准备接收数据
        // UDP协议中 一个数据报最大为64k
         buf = new byte[64 * 1024];
        DatagramPacket packet = new DatagramPacket(buf, buf.length);
        // 3.调用接收方法，将通过网络传输的DatagramPacket对象
        //   填充到 packet对象
        //   当有数据报发送到本机的9999端口时，就会接受数据
        //   如果没有数据报发送到本机的9999端口，就会阻塞等待
        System.out.println("接收端A 等待接收数据~");
        socket.receive(packet);

        // 4.可以把packet进行拆包，取出数据并显示
        int length = packet.getLength(); //实际接收的数据字节长度
        byte[] data = packet.getData();
        String s = new String(data, 0, length);
        System.out.println(s);

        buf = "ok，我带女友去~".getBytes();
        DatagramPacket packet1 =
                new DatagramPacket(buf, buf.length, InetAddress.getLocalHost(), 9998);
        socket.send(packet1);

        socket.close();
        System.out.println("A端退出");
    }
}

// 发送端B
public class B {
    public static void main(String[] args) throws IOException {
        byte[] data = null;
        // 1.创建 DatagramSocket对象，准备发送和接收数据
        DatagramSocket socket = new DatagramSocket(9998);

        // 2.将需要发送的数据，封装到DatagramPacket对象
        data = "hello 明天吃火锅".getBytes();

        // 说明：封装的 DatagramPacket对象 data 内容字节数组，data.length,主机（IP），端口
        DatagramPacket packet =
                new DatagramPacket(data, data.length, InetAddress.getLocalHost(), 9999);
        socket.send(packet);

        data = new byte[64 * 1024];
        DatagramPacket packet1 = new DatagramPacket(data, data.length);
        socket.receive(packet1);
        int length = packet1.getLength();
        byte[] data1 = packet1.getData();
        String s = new String(data1, 0, length);
        System.out.println(s);

        // 关闭资源
        socket.close();
        System.out.println("B端退出");
    }
}
