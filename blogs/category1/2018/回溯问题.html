    组合问题没啥难的，无非就是需要进行这个去重的判断，去重判断之前首先要进行这个数组的排序；就是避免这个情况即可 if(i > 1 && !used[i - 1] && nums[i] == nums[i - 1]) 则 跳过此次的循环即可；

    电话号码的字母组合，之前的题目都是在本集合中一个数来搭配其他数进行操作的，而这个题目是本集合的数搭配另一个集合的数来进行操作的；

    class Solution {
    List<String> res = new LinkedList<>();
    StringBuilder sb = new StringBuilder();

    public List<String> letterCombinations(String digits) {
        if (digits.length() == 0) {
            return res;
        }
        String[] numString = {"", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};
        //迭代处理
        back(digits, numString, 0);
        return res;
    }

    public void back(String digits, String[] numString, int index) {
        // 终止条件：index代表的是指向哪个数字的索引
        if (index == digits.length()) {
            res.add(sb.toString());
            return;
        }
        // 选择的是哪个数字
        int num = digits.charAt(index) - '0';
        // 获取到字符串
        String letter = numString[num];
        // for (选择：其他集合的元素)
        for (int i = 0; i < letter.length(); i++) {
            sb.append(letter.charAt(i));
            back(digits, numString, index + 1);
            sb.deleteCharAt(sb.length() - 1);
        }
    }
}

    # 排列问题

    • [ ] 全排列
    • [ ] 全排列II

    两道题目都是比较简单的题目，画图直接可以看出答案的，比较直观；

    全排列就是每次从 0 开始进行遍历，然后如果遇到纵列相同的值，即就是自己嘛，就跳过； if(used[i]) continues;

    全排列II 也是从 0 开始遍历，然后如果遇到纵列相同的值其实就是本身，而且横向看前后两个值相同都需要进行跳过；即在全排列基础上再加上一行代码：

    if(i > 0 && !used[i - 1] && nums[i - 1] == nums[i]) continues;即可。

    // 全排列II
class Solution {
    LinkedList<Integer> list = new LinkedList<>();
    List<List<Integer>> res = new LinkedList<>();

    public List<List<Integer>> permute(int[] nums) {
        Arrays.sort(nums);
        back(nums, new boolean[nums.length], 0);
        return res;
    }

    public void back(int[] nums, boolean[] used, int count) {
        if (count == nums.length) {
            res.add(new LinkedList<>(list));
            return;
        }

        for (int i = 0; i < nums.length; i++) {
            // 如果是同一纵列，且使用过的就跳过
            if (used[i]) {
                continue;
            }
            // 如果是同一横行，且值相同的就跳过
            if (i > 0 && !used[i - 1]  && nums[i] == nums[i - 1]) {
                continue;
            }
            used[i] = true;
            list.add(nums[i]);
            count++;
            back(nums, used, count);
            count--;
            list.removeLast();
            used[i] = false;
        }
    }
}

    # 字符串切割问题

    • [ ] 分割回文子串（二刷）

    • [ ] 复原 IP 地址（二刷）

    分割回文字串：

    做这个题目的时候，需要思考这题目有哪些难点：

    ①哪里是切割点，比如说："aab" ，我们需要将 (0, startIdx,i) 的这个字串扔进去进行判断；

    ②如果是回文子串，就需要进行切割存储处理；s.substring(startIdx,i + 1)；

    ③回溯；

    ④判断是否为回文串

    class Solution {
    LinkedList<String> list = new LinkedList<>();
    List<List<String>> res = new LinkedList<>();

    public List<List<String>> partition(String s) {
        backTracking(s, 0);
        return res;
    }
    public void backTracking(String s, int index) {
        // 终止条件
        if (s.length() <= index) {
            res.add(new LinkedList<>(list));
            return;
        }

        for (int i = index; i < s.length(); i++) {
            if (isHuiWen(s, index, i)) {
                // 切割需要算清楚 i + 1
                list.add(s.substring(index, i + 1));
                backTracking(s, i + 1);
                if (!list.isEmpty()) {
                    list.removeLast();
                }
            }
        }
    }
    public boolean isHuiWen(String s, int i, int j) {
        if (i > j) {
            return false;
        }
        while (i < j) {
            if (s.charAt(i++) != s.charAt(j--)) {
                return false;
            }
        }
        return true;
    }
}

    N皇后 & N皇后II

    难点：

    ①我们需要自己自定义一个二维数组，其中用 "." 来填充；

    ②对于 N 皇后，啥时候进行结果的收集呢？ ==> 当我们的 row 值等于我们的节点深度时就是我们收集结果的时候；

    ③N皇后的规则需要界定：不能同行同列同斜线

    class Solution {
    List<List<String>> res = new LinkedList<>();

    public List<List<String>> solveNQueens(int n) {
        String[][] arr = new String[n][n];
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr.length; j++) {
                arr[i][j] = ".";
            }
        }
        backTracking(n, 0, arr);
        return res;
    }

    public void backTracking(int n, int row, String[][] arr) {
        // 当我们树的高度等于这个row 的时候 就是收集节点的时候
        if (row == arr.length) {
            LinkedList<String> list = new LinkedList<>();
            for (int i = 0; i < n; i++) {
                StringBuilder sb = new StringBuilder();
                for (int j = 0; j < n; j++) {
                    sb.append(arr[i][j]);
                }
                list.add(sb.toString());
            }
            res.add(list);
            return;
        }

        for (int i = 0; i < n; i++) {
            if (isNQueen(row, i, arr, n)) {
                arr[row][i] = "Q";
                backTracking(n, row + 1, arr);
                arr[row][i] = ".";
            }
        }
    }
    public boolean isNQueen(int row, int col, String[][] arr, int n) {
        // 纵列
        for (int i = 0; i < n; i++) {
            if ("Q".equals(arr[i][col])) {
                return false;
            }
        }
        // 45度
        for (int i = row - 1, j = col + 1; i >= 0 && j < n; i--, j++) {
            if ("Q".equals(arr[i][j])) {
                return false;
            }
        }
        // 135度
        for (int i = row - 1, j = col - 1; i >= 0 && j >= 0; i--, j--) {
            if ("Q".equals(arr[i][j])) {
                return false;
            }
        }
        return true;
    }
}

    解数独

    这道题目不勉强，能看懂就好了；

    class Solution {
    public void solveSudoku(char[][] board) {
        back(board);
    }

    public boolean back(char[][] board) {
        for (int i = 0; i < board.length; i++) {
            for (int j = 0; j < board.length; j++) {
                if (board[i][j] != '.') {
                    continue;
                }
                // 将 1 - 9 尝试着来进行填入矩阵中
                for (char k = '1'; k <= '9'; k++) {
                    if (isShuDu(i, j, k, board)) {
                        board[i][j] = k;
                        if (back(board)) {
                            return true;
                        }
                        board[i][j] = '.';
                    }
                }
                // 九个数都用完了都不合适，证明这个不行 数独不行没有解
                return false;
            }
        }
        return true;
    }

    public boolean isShuDu(int row, int col, int val, char[][] board) {
        // 纵列
        for (int i = 0; i < board.length; i++) {
            if (val == board[i][col]) {
                return false;
            }
        }

        // 横列
        for (int i = 0; i < board.length; i++) {
            if (val == board[row][i]) {
                return false;
            }
        }

        int n = row / 3;
        int m = col / 3;
        for (int i = n * 3; i < (n + 1) * 3; i++) {
            for (int j = m * 3; j < (m + 1) * 3; j++) {
                if (board[i][j] == val) {
                    return false;
                }
            }
        }
        return true;
    }
}

    这题目看起来知道这个做法，但是写起来各种坑，其实这道题目和图论中的岛屿数量有比较相同的地方的，无非就是从一个字母开始进行暴力求解，看是否能找到符合的字符串，写题目的时候我一直有一个疑惑，就是该不该写这个返回值呢？其实我们可以参考一下二叉树寻找一条路径的时候的情景：这道题目不需要遍历这个二叉树，我们只需要遍历到符合条件的字符串后就可以进行返回了，那么我们就需要返回值！

    class Solution {
    private static final int[][] DIRECTIONS = {{-1, 0}, {0, -1}, {0, 1}, {1, 0}};
    /**
     * 行
     */
    private int rows;
    /**
     * 列
     */
    private int cols;
    /**
     * word的长度
     */
    private int len;
    /**
     * 标记数组
     */
    private boolean[][] visited;
    /**
     * word转换为char数组
     */
    private char[] charArray;
    /**
     * 入参中的board数组
     */
    private char[][] board;

    public boolean exist(char[][] board, String word) {
        rows = board.length;
        if (rows == 0) {
            return false;
        }
        cols = board[0].length;
        visited = new boolean[rows][cols];

        this.len = word.length();
        this.charArray = word.toCharArray();
        this.board = board;
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                // 每个字符都需要进行一次深搜
                if (dfs(i, j, 0)) {
                    return true;
                }
            }
        }
        return false;
    }

    /**
     * @param begin 索引来的，如果该索引等于我们的 len - 1
     */
    private boolean dfs(int x, int y, int begin) {
        if (begin == len - 1) {
            return board[x][y] == charArray[begin];
        }
        if (board[x][y] == charArray[begin]) {
            visited[x][y] = true;
            for (int[] direction : DIRECTIONS) {
                int newX = x + direction[0];
                int newY = y + direction[1];
                // 不能越界 不能访问已经走过的
                if (inArea(newX, newY) && !visited[newX][newY]) {
                    if (dfs(newX, newY, begin + 1)) {
                        return true;
                    }
                }
            }
            // 回溯
            visited[x][y] = false;
        }
        return false;
    }

    private boolean inArea(int x, int y) {
        return x >= 0 && x < rows && y >= 0 && y < cols;
    }
}
