    移除元素

    考小拉 7/18/2022 算法数组双指针
    • 内容摘抄代码随想录

    # 基础知识：

    1. 数组是存放在连续内存空间上的相同数据类型的集合

    2. 数组下表都是从0开始的

    3. 数组的内存地址都是连续的；二维数组如int[][] arr = new int [3][4];可不是一个连续的3*4的连续地址空间

    所以Java的二维数组在内存中不是 3*4 的连续地址空间，而是四条连续的地址空间组成！

    4.因为一维数组的空间是连续的，在插入或者删除元素时候难免会产生移动其他元素的行为

    力扣题目链接

    # 代码展示

    //暴力解法
public static int MoveElement(int[] arr, int target) {
        int size = arr.length;
        for (int i = 0; i < size; i++) {
            if (arr[i] == target) {
                for (int j = i + 1; j < size; j++) {
                    arr[j - 1] = arr[j];
                }
                i--; //因为下标i以后的数组都向前移动一位，所以i也要向前移动一位
                size--; //数组的长度要-1
            }
        }
        return size;
    }

    定义快慢指针

    • 快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组
    • 慢指针：指向更新 新数组下标的位置
    // 双指针法
public static int MoveElementplus(int[] arr, int target) {
        int fastIndex = 0;
        int slowIndex;
        for (slowIndex = 0; fastIndex < arr.length; fastIndex++) { //int[] arr1 = {0,1,2,2,3,4,5,2,7};
            if (arr[fastIndex] != target) {
                arr[slowIndex] = arr[fastIndex];
                slowIndex++;
            }
        }
        return slowIndex;
    }
