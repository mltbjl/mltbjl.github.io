    动态规划-子序列 专题

    考小拉 9/2/2023 数据结构与算法hot 100 & 牛客 系列

    # 连续问题

    • [ ] 最长连续递增序列
    • [ ] 最长重复子数组
    • [ ] 最大子序和

    最长连续递增序列中，我们设定 dp 数组的含义：到达索引 i - 1 为结尾的字符，最长的连续递增序列为 dp[i]；剩下的就是递推公式了：如果是当前数大于前一个数，就： dp[i] = Math.max(dp[i - 1] + 1, dp[i]); 否则就是不理；

    class Solution {
    public int findLengthOfLCIS(int[] nums) {
		int[] dp = new int[nums.length];
        Arrays.fill(dp, 1);
        int result = 0;
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] > nums[i - 1]) {
                // dp[i - 1]数组的含义就是以 i - 1 为结尾的字符的最长的连续递增子序列
                dp[i] = Math.max(dp[i - 1] + 1, dp[i]);
            }
            if (result < dp[i]) {
                result = dp[i];
            }
        }
        return result == 0 ? 1 : result;
    }
}

    最大子序和，这道题可以和我们上面的 最长连续递增序列 进行比较：

    我们需要确定好这个 dp 数组的含义：以 i - 1 为结尾的字符的最长的连续的子序和为 dp[i] ，我们的递推公式为：dp[i] = Math.max(dp[i - 1] + nums[i], dp[i]);

    public class Solution {
    public int FindGreatestSumOfSubArray (int[] array) {
        // write code here
        if (array.length == 1) {
            return array[0];
        }
        // dp数组的含义：以i为结尾的元素，连续子数组的最大值为dp[i]
        int[] dp = new int[array.length];
        // 初始化
        for (int i = 0; i < array.length; i++) {
            dp[i] = array[i];
        }
        int maxVal = array[0];
        for (int i = 1; i < array.length; i++) {
            if (dp[i - 1] + array[i] > 0) {
                // dp[i - 1]的含义是：以i - 1为结尾的字符的连续子数组最大值为dp[i - 1]
                dp[i] = Math.max(dp[i], dp[i - 1] + array[i]);
            }
            if (dp[i] > maxVal) {
                maxVal = dp[i];
            }
        }
        return maxVal;
    }
}

    最长重复子数组的话，就难度不大了，我们直接可以画出这个二维数组出来推演即可；

    综上，连续的子序列问题的一般做法是：

    1. dp 数组的定义十分地重要，一般来说都是 这样定义得：问啥定义啥就行了（一般都是在以下标为i为结尾的数前最大的 ** 为dp[i]）；
    2. 关于二维数组的连续的子序列问题也可以通过画图的形式进行推演；
    3. 因为是连续子序列，所以多少都是跟 dp[i - 1] 有关系的；同时需要注意的是：假如求的是 个数 次数，那么一般都是 dp[i - 1] + 1 来做这个比较； 但是如果是求 值的最大最小值，就是 dp[i - 1] + nums[i]；（参考 最长连续递增序列 和 最大子序和）
    4. 而且在处理二维数组的问题的时候，无关的数可以不需要处理（参考 最长重复子数组）

    # 非连续问题

    • [ ] 最长上升（递增）子序列

    • [ ] 最长公共子序列

    • [ ] 不相交的线

    • [ ] 最长递增子序列的个数（没懂）

    综上四个题目可以看出 非连续的子序列问题的做法：

    1. dp 数组的定义十分地重要，一般来说都是 这样定义得：dp[i] 表示 i 之前包括i的以 nums[i] 结尾的最长递增子序列的长度

    2. 第一个如果是可以对元素进行删减的，那么一般来说都是需要两层for循环的，第一层for循环肯定是用来遍历整个数组的，主要是第二层for循环，基于dp数组的定义，那么基础框架就是这样子的了。（比如第一题）

    3. 处理二维数组的时候，即使是无关的数都是需要进行处理的；（这里可以参照对比一下 最长公共子序列 和 最大子序和）

    # 编辑距离问题

    1. 判断子序列
    2. 不同的子序列
    3. 两个字符串的删除操作
    4. 编辑距离

    综上，关于编辑距离的题目，得出来的总体技巧：

    1. 最重要的还是要确认好dp数组的含义：比如拿【不同的子序列】来做例子：dp数组的含义就是 以i - 1为结尾的s子序列包含 j - 1为结尾的t子序列个数为 dp[i] [j]。
    2. 同时，一般编辑距离的问题都是可以用二维数组来进行推算解决的（假如遇到字符相同的时候如何；不同的时候又如何）；

    补充：dp数组的含义：

    1. 判断子序列：以 i - 1 为结尾的字符串t包含了以 j - 1 为结尾的s的序列为dp[i - 1][j - 1]
    2. 不同的子序列：以 i - 1 为结尾的字符串s包含了以 j - 1 为结尾的t的个数为dp[i - 1][j - 1]
    3. 两个字符串的删除操作：以 i - 1 为结尾的字符串word1和以 j - 1 为结尾的word2相同需要的最小删除操作数dp[i - 1][j - 1]
    4. 编辑距离：以 i - 1 为结尾的字符串word1变成以 j - 1 为结尾的word2所需要的最少操作数为dp[i - 1][j - 1]
