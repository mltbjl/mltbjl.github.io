    • 汽水瓶换饮料 (opens new window)

    这题目就是用暴力的方式进行解决的，主要是思路需要理清楚。首先我们直接讲最重点的那一部分吧，三个空瓶子换一瓶汽水，假如我有10个空瓶子，这个推导的过程是怎么样的？

    10个空瓶子 --> 换3瓶汽水 + 剩余一个空瓶子 --> 那么此时就是 “四个” 空瓶子了 --> 重复上面的流程，换一瓶汽水 + 剩余一个空瓶子 --> 最后只剩下两个空瓶子了，题目又说可以像店家借一个空瓶子，所以又是可以达到了三个空瓶子了 ---> 又换到一个汽水，最后把这个空瓶子还给店家。

    我们需要关注这个循环的过程，这也是这个题目的核心重点。

    public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        while (sc.hasNextLine()) {
            int num = Integer.parseInt(sc.nextLine());
            if (num == 0) {
                return;
            } else if (num < 2) {
                System.out.println(0);
            } else if (num == 2) {
                System.out.println(1);
            } else {
                int res = 0, count;
                // 核心
                while (num > 2) {
                    count = num / 3;
                    num = num % 3;
                    res += count;
                    num = count + num;
                }
                // 额外的瓶子，如果等于2，证明还可以兑换一瓶（向店家要一个瓶子）
                if (num == 2) {
                    res += 1;
                }
                System.out.println(res);
            }
        }
    }
}

    • 开发商购买土地 (opens new window)

    这道题目第一眼看的时候以为是 dp 数组的题目，后面其实可以使用暴力法；我们要求的是切割后的权值嘛，所以我们就是需要找到一个权值差最小的即可；那么我们顺着这个思路：

    第一种先按行进行遍历，累计两部分的权值，然后进行计算权值差；第二种则按照列进行遍历，也是累计两部分的权值，然后进行计算权值差；最后在比较两个权值差即可；

    public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String[] s = sc.nextLine().split(" ");
        int m = Integer.parseInt(s[0]);
        int n = Integer.parseInt(s[1]);
        int[][] arr = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                arr[i][j] = sc.nextInt();
            }
        }

        System.out.println(get(arr));
    }

    public static int get(int[][] arr) {
        // 计算 sum 值
        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[0].length; j++) {
                sum += arr[i][j];
            }
        }
        int m = arr.length;
        int n = arr[0].length;
		// 按行算的两部分权值
        int count1 = 0;
        int count2 = 0;
        // 权差1 2
        int res1 = Integer.MAX_VALUE, res2 = Integer.MAX_VALUE;
        for (int i = 0; i < m - 1; i++) {
            for (int j = 0; j < n; j++) {
                count1 += arr[i][j];
                count2 = sum - count1;
            }
            res1 = Math.min(Math.abs(count1 - count2), res1);
        }
		// 按列算的两部分权值
        int count3 = 0;
        int count4 = 0;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < m; j++) {
                count3 += arr[j][i];
                count4 = sum - count3;
            }
            res2 = Math.min(res2, Math.abs(count4 - count3));
        }
        // 比较行列的权值差的大小
        return Math.min(res1, res2);
    }
}
