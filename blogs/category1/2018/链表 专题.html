    # 移除链表元素（可以用双指针）

    力扣题目链接 (opens new window)

    注意： 可以使用原链表来直接进行删除操作，也可以设置一个虚拟头结点来进行删除操作。

    该题目简单，就用来回顾一下用原链表删除结点操作：

    比如 链表为： 1 -> 4 -> 2 -> 4 移除1；也就是移除头结点

    移除头结点的操作和移除其他结点操作是不一样的，链表的其他结点都是通过前一个结点来移除当前结点，但是头结点没有前一个结点 ---> 那么就将头结点后移即可（从 1 换成为 4，4为头结点）。

    // 原链表删除结点操作
public ListNode removeElements(ListNode head, int val) {
    if(head == null) return null;
    // 头结点如果是需要删除的结点，单独写一段逻辑
    if(head.val == val)  head = head.next;
    ListNode pre = head;
    ListNode cur = head.next;
    while(cur != null) {
        if(cur.val == val) {
            pre.next = cur.next;
        }else {
            pre = cur;
        }
        cur = cur.next;
    }
    return head;
}

    因为上述中用原链表直接进行删除操作，需要单独给处理移除头结点的情况，那是否有统一移除结点的办法？

    移除其他结点操作：设置一个虚拟头结点，这样原链表的所有结点都可以按照统一的方式进行删除了。

    // 设置虚拟头结点来操作
public ListNode removeElements(ListNode head, int val) {
    if(head == null) return null;
    // 设置虚拟头结点，原链表的所有结点统一处理
    ListNode dummy = new ListNode(-1,head);
    ListNode pre = dummy;
    ListNode cur = head;
    while(cur != null) {
         if (cur.val == val) {
            pre.next = cur.next;
        } else {
            pre = cur;
        }
        cur = cur.next;
    }
    return dummy.next;
}
/*
时间复杂度：O(n)
空间复杂度：O(1)
*/

    我们打算在原有的链表上操作即可：其实这道题目涉及到三个结点的操作，用双指针的进行操作。最重要就是转换的逻辑

    pre = null;

    class ListNode{
    int val;
    ListNode next;
    public ListNode(){}
    public ListNode(int val) { this.val = val; }

public ListNode reverseList(ListNode head) {
        ListNode prev = null;
        ListNode cur = head;
        ListNode temp = null;
        while (cur != null) {
            temp = cur.next; // 保存下一个节点
            cur.next = prev;
            prev = cur;
            cur = temp;
        }
        return prev;
    }
/*

    # 两两交换链表中的节点

    力扣题目链接 (opens new window)

    做完上一道题目后，这道题目的操作结点也是如出一辙。建议用虚拟头结点配合操作。

    public ListNode swapPairs(ListNode head) {
    ListNode dummyHead = new ListNode(-1);
    dummy.next = head;
    ListNode cur = dummyHead;
    ListNode temp; // 临时节点，保存两个结点的后面那个结点
    ListNode firstNode; // 临时节点，保存两个节点之间的第一个节点
    ListNode secondNode; // 临时节点，保存两个节点之间的第二个节点
    // 奇数个节点 或者 偶数个节点
    while(cur.next != null && cur.next.next != null) {
        temp = cur.next.next.next;
        firstNode = cur.next;
        secondNode = cur.next.next;
        
        cur.next = secondNode;
        secondNode.next = firstNode;
        firstNode.next = temp;
        cur = firstNode; // cur移动，准备下一轮交换
    }
    return dummyHead.next;
}
/*
时间复杂度：O(n*m) m 代表题目中的 n，即倒数第n个元素
空间复杂度：O(1)
*/

    # 删除链表倒数第n个元素

    力扣题目链接 (opens new window)

    本人的做法：直接先计数全部的结点数量，然后再去遍历 n - 1次，利用 cur.next = cur.next.next;即可略过倒数第n个结点。

    public ListNode removeNthFromEnd(ListNode head, int n) {
        if(head.next == null) return null;
        ListNode dummyHead = new ListNode(-1);
        dummyHead.next = head;
        ListNode listNode = dummyHead; // 先存着dummyHead结点
        int count = 0;
        while (dummyHead != null) {
            count++;
            dummyHead = dummyHead.next;
        }
        dummyHead = listNode; // 重置dummyHead为虚拟头结点
        for (int i = 0; i < count - n; i++) {
            if(i == count - n - 1) {
                dummyHead.next = dummyHead.next.next;
                break;
            }
            dummyHead = dummyHead.next;
        }
        return listNode.next;
    }
/*
时间复杂度：O(n*m) m 代表题目中的 n，即倒数第n个元素
空间复杂度：O(1)
*/

    # 链表相交

    力扣题目链接 (opens new window)

    简单来说，就是求两个链表交点的节点的指针。交点不是数值相等，而是指针相等。

    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
    ListNode curA = headA;
    ListNode curB = headB;
    int lenA = 0,lenB = 0;
    while(curA != null) { // 求链表A的长度
        lenA++;
        curA = curA.next;
    }
    while(curB != null) { // 求链表B的长度
        lenB++;
        curB = curB.next;
    }
    curA = headA;
    curB = headB;
    // 让curA为最长链表的头，lenA为最长链表的长度
    if(lenB > lenA) {
        // swap(lenA,lenB)
        int tmpLen = lenA;
        lenA = lenB;
        lenB = tmpLen;
        // swap(curA,curB)
        ListNode tmpNode = curA;
        curA = curB;
        curB = tmpNode;
    }
    // 求长度差
    int gap = lenA - lenB;
    // 让curA与curB在同一起点上（末尾位置对齐）
    while(gap-- > 0) {
        curA = curA.next;
    }
    // 遍历curA和curB，遇到相同的直接返回
    while(curA != null) {
        if(curA == curB) {
            return curA;
        }
        curA = curA.next;
        curB = curB.next;
    }
    return null;
}

    # 回文链表

    力扣题目链接 (opens new window)

    最直接的想法就是 将链表的节点数遍历出来，然后开辟相应大小的数组空间，然后在数组中判断是否回文即可。

    public boolean isPalindrome(ListNode head) {
        if(head.next == null) return true;
        int count = 0;
        ListNode temp = head;
        while(temp != null) {
            count++;
            temp = temp.next;
        }
        int[] arr = new int[count];
        temp = head;
        for(int i = 0;i < count;i++) {
            arr[i] = temp.val;
            temp = temp.next;
        }
        for(int i = 0,j = count - 1; i < j; i++ , j--) {
            if(arr[i] != arr[j]) return false;
        }
        return true;
    }
/*
时间复杂度：O(n)
空间复杂度：O(n)
*/

    # 环形链表（可以用双指针）

    力扣题目链接 (opens new window)

    使用快慢指针；快指针fast 从头节点出发，每次移动两个节点，slow指针从头节点出发，每次移动一个节点，如果有环，肯定会相遇。（可以自己画图验证）

    public boolean hasCycle(ListNode head) {
    ListNode fast = head;
    ListNode slow = head;
    while(fast != null && fast.next != null) {
        fast = fast.next.next; // 快指针
        slow = slow.next; // 慢指针
        if(fast == slow) {
            return true;
        }
    }
    return false;
}
/*
时间复杂度：O(n)
空间复杂度：O(1)
*/

    # 环形链表II（可以用双指针）

    力扣题目链接 (opens new window)

    • 判断链表是否环（上一道题已经解决）
    • 如果有环，如何找到这个环的入口；那么如何找到该环的入口？在相遇节点上定义一个指针index1，在头结点定义一个指针index2 ，两个指针每次都走一个节点；如果index1 == index2，那么就是环的入口。（卡哥这里是数学推导自行去推吧，我这里记住结论即可）
    public ListNode hasCycle(ListNode head) {
    ListNode fast = head;
    ListNode slow = head;
    while(fast != null && fast.next != null) {
        fast = fast.next.next; // 快指针 每次移动两个节点
        slow = slow.next; // 慢指针 每次移动一个节点
        if(fast == slow) {
            ListNode index1 = head;;
            ListNode index2 = slow;
            while(index1 != index2) {
                index1 = index1.next;
                index2 = index2.next;
            }
            return index1;
        }
    }
    return null;
}
/*
时间复杂度：O(n*m)
空间复杂度：O(1)
*/

    public ListNode add(ListNode l1,ListNode l2) {
        int carry = 0;
        ListNode sumHead = new ListNode(-1);
        ListNode sumNode = sumHead;
        while (l1 != null || l2 != null) {
            // 求和
            int x = l1 != null ? l1.val : 0;
            int y = l2 != null ? l2.val : 0;
            int sum = x + y + carry;
            if (sum > 10) {
                carry = 1;
                sum = sum - 10;
            } else {
                carry = 0;
            }
            sumNode.next = new ListNode(sum);
            // 三个指针后移
            l1 = l1 != null ? l1.next : null;
            l2 = l2 != null ? l2.next : null;
            sumNode = sumNode.next;
        }
        if (carry != 0) {
            sumNode.next = new ListNode(carry);
        }
        return sumHead.next;
    }
/*
时间复杂度：O(n)
空间复杂度：O(1)
*/

    # 合并两个有序的链表

    力扣题目链接 (opens new window)

    这也是一道操作两个链表的题目；

    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        ListNode dummyNode = new ListNode(0);
        ListNode cur = dummyNode;
   		
        while (list1 != null && list2 != null) {
            if (list1.val < list2.val) {
                cur.next = list1;
                cur = cur.next;
                list1 = list1.next;
            } else {
                cur.next = list2;
                cur = cur.next;
                list2 = list2.next;
            }
        }
        if (list1 == null) {
            cur.next = list2;
        } else {
            cur.next = list1;
        }
        return dummyNode.next;
    }
/*
时间复杂度：O(n)
空间复杂度：O(1)
*/

    # 总结

    1. 链表题目多数都可以往双指针方面靠，毕竟需要操作结点方便；
    2. 建议在删除类的题目或者是需要单独操作头结点的题目时，都用虚拟头结点代替链表原本的头结点，这样的话不需要单独写一个逻辑来处理头结点的问题；
    • 移除链表元素（可以用双指针）
    • 翻转链表（可以用双指针）
    • 两两交换链表中的节点
    • 删除链表倒数第n个元素
    • 链表相交
    • 回文链表
    • 环形链表（可以用双指针）
    • 环形链表II（可以用双指针）
    • 两数相加
    • 合并两个有序的链表
